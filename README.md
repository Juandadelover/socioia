# SocioIA Landing Page

Premium landing page for SocioIA - AI-powered business solutions in Valledupar, Colombia.

## Tech Stack

- React 18
- Vite
- Framer Motion
- React Hook Form
- Formspree

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
